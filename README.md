# Study Sheet Generator

A client-only React app to convert notes into A4-printable study sheets, export to PDF, and share via short URL (state encoded in URL hash).

Main features:
- Editor + live Preview
- One-column & two-column templates
- A4 PDF export using html2canvas + jsPDF
- Auto-save (localStorage)
- Shareable compressed URL (lz-string)
- Accessible (labels, aria attrs)
- Deployable on Netlify / Vercel / GitHub Pages

See implementation details in the full project files.
